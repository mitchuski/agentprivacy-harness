## How work is actually done here

Two independent instances of this pattern — the papers pipeline and a ZK circuit workshop — converged on the same shape without borrowing from each other. `[R·I·🔗 ← agentprivacy-docs/papers/Programme/pipeline/chronicles/2026-07-09_dual-agent-harness-brief.md]`

1. **One session, one seat, one card.** `[P·I·🛡️ ← CLAUDE.md "One session holds exactly one seat"; every seats/*.md Reads list]`
2. **Shared state has exactly one writer;** ledgers are append-only. `[P·I·🛡️ ← GROUND_RULES.md GR-10]`
3. **A gate ladder ending in a rung no agent may climb.** The papers pipeline marks `p4_owner: first-person`; the ZK workshop calls it `Z4`; this repo's own template marks `g4_owner: first-person` on `G4-door`. `[P·I·🛡️ ← agentprivacy-docs/papers/Programme/pipeline/manifest.yaml:10; tig_zk_loop/HANDOFF_PRIVACY_POOLS_V2.md:137; templates/manifest.yaml:9,11]`
4. **Adversarial review is a seat, and its findings are contracts** — reads the artifact alone, no canon access, no goodwill, prices findings by severity, never edits. `[P·I·🪞 ← agentprivacy-docs pipeline chronicle 2026-07-02_the-day-of-three-cycles.md: "no canon access and no goodwill"]`
5. **Interruption means resume, never rebuild** — verify disk state before trusting a prior verdict. `[P·I·🪞 ← agentprivacy-docs pipeline chronicles, "the resume-not-rebuild pattern/rule" (2026-07-09, 2026-07-10)]`
6. **Failures are loud; negative results are filed at win-prominence.** A declared floor in the ZK workshop (CR-10) died in hours to a further pass, filed as prominently as a win. `[P·I·🪞 ← GROUND_RULES.md GR-6; tig_zk_loop/HANDOFF_PRIVACY_POOLS_V2.md:93 "CR-10 died in hours"]`

**Where this applies:** the harness transfers to any pipeline where generated artifacts must survive adversarial scrutiny before a human commits to them — port the rules, rename the roles, keep the hard stop. `[P·I·🧙 ← agentprivacy-docs/papers/Programme/pipeline/chronicles/2026-07-09_dual-agent-harness-brief.md:162-166]`

The deeper test, from Promise Theory: the split is warranted only when one agent would otherwise have to promise two things it cannot independently control — protection *and* delegation. `[D·S·🛡️ ← TRUSTS.md T1, the Autonomy Axiom]`
